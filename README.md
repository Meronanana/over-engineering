## Getting Started

프로젝트를 실행하기 위해 아래 명령어를 터미널에 입력하세요.

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 를 열어 결과물을 확인하세요.

버그 없는 최신 작품을 확인하고 싶다면, develop 브랜치를 이용하세요.
